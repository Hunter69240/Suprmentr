# 1️⃣ Removing Duplicates in Pandas

## 📌 Definition
Removing duplicate rows means deleting rows that are completely identical across all columns.

## ✅ Basic Syntax
```python
df = df.drop_duplicates()
```

## 🔎 Key Points
- Removes rows only if **all column values match exactly**
- String comparison is **case-sensitive**
- NaN values are treated as equal for duplicate detection

## 🎯 Remove Duplicates Based on Specific Columns
```python
df = df.drop_duplicates(subset=['age', 'city'])
```

This removes duplicates considering only selected columns.

---

# 2️⃣ Standardizing City Names (Text Normalization)

Standardizing text means making string values consistent in format.

⚠ Do NOT use StandardScaler or MinMaxScaler for text data.

## ✅ Convert to Lowercase
```python
df['city'] = df['city'].str.lower()
```

## ✅ Remove Extra Spaces
```python
df['city'] = df['city'].str.strip()
```

## ✅ Combine Both
```python
df['city'] = df['city'].str.strip().str.lower()
```

## ✅ Convert to Proper Case
```python
df['city'] = df['city'].str.strip().str.title()
```

## ✅ Fix Known Spelling Variations
```python
city_map = {
    'banglore': 'bangalore',
    'bengaluru': 'bangalore'
}

df['city'] = df['city'].replace(city_map)
```

# 3️⃣ Standardization (StandardScaler)

## 📌 Definition
Standardization transforms numeric data so that:
- Mean = 0
- Standard Deviation = 1

## 🧮 Formula
```
z = (x - μ) / σ
```

Where:
- x = original value
- μ = mean
- σ = standard deviation

## ✅ Implementation
```python
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
df[['age','salary']] = scaler.fit_transform(df[['age','salary']])
```

## 🎯 When to Use
- Logistic Regression
- SVM
- KNN
- Neural Networks

---

# 4️⃣ Normalization (MinMaxScaler)

## 📌 Definition
Scales numeric values between 0 and 1.

## 🧮 Formula
```
X_scaled = (X - X_min) / (X_max - X_min)
```

## ✅ Implementation
```python
from sklearn.preprocessing import MinMaxScaler

scaler = MinMaxScaler()
df[['age','salary']] = scaler.fit_transform(df[['age','salary']])
```

## ⚠ Important
MinMaxScaler does NOT handle NaN values.

### 🔧 Handle Missing Values First

Option 1: Fill missing values
```python
df['age'] = df['age'].fillna(df['age'].mean())
df['salary'] = df['salary'].fillna(df['salary'].mean())
```

Option 2: Drop missing rows
```python
df = df.dropna(subset=['age','salary'])
```

---

# 5️⃣ Standardization vs Normalization

| Feature | StandardScaler | MinMaxScaler |
|----------|----------------|---------------|
| Output Range | Not fixed | 0 to 1 |
| Mean | 0 | Not fixed |
| Std Dev | 1 | Not fixed |
| Sensitive to Outliers | Less | More |
| Common Use | Most ML models | Bounded features |

---

# 6️⃣ Data Type vs Technique

| Data Type | Technique |
|------------|-----------|
| Text | String normalization |
| Categorical | Encoding (OneHot / Label Encoding) |
| Numeric | Scaling (StandardScaler / MinMaxScaler) |
| Duplicate Rows | drop_duplicates() |

---

# 7️⃣ Recommended Preprocessing Order

1. Handle missing values  
2. Standardize text columns  
3. Remove duplicates  
4. Encode categorical variables  
5. Scale numerical features  

---

# 🎯 Final Summary

- drop_duplicates() → Removes identical rows  
- String methods (.lower(), .strip(), .title()) → Standardize text  
- StandardScaler → Mean = 0, Std = 1  
- MinMaxScaler → Values between 0 and 1  
- Always handle missing values before scaling
